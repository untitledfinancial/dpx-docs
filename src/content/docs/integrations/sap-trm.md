This integration guide is coming soon.

SAP TRM customers will be able to trigger DPX settlements from within SAP payment runs, with ESG-weighted fee calculation and on-chain settlement confirmation written back to SAP.

## Planned capabilities

- DPX as a payment method within SAP payment programs
- ESG score lookup and fee pre-calculation before posting
- Settlement confirmation and transaction hash stored in SAP
- Compliance-ready on-chain audit trail
