Kyriba customers will be able to route settlements through DPX directly from their treasury dashboard, with live ESG scoring and on-chain fee verification.

## Planned capabilities

- Price DPX settlements from within Kyriba payment workflows
- ESG score lookup before settlement confirmation
- Automated settlement execution via Kyriba API triggers
- On-chain audit trail linked to Kyriba transaction records
