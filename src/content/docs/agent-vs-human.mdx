## How the agent executes this

In a production treasury agent, the full loop looks like this — no human involved:

```python
@tool
def dpx_settle(amount_usd: float, currency_pair: str) -> dict:
    """Check oracle, quote, verify, and execute a DPX settlement."""
    health = httpx.get("https://stability.untitledfinancial.com/reliability").json()
    if not health["isHealthy"]:
        return {"status": "deferred", "reason": "oracle instability"}
    quote = httpx.get("https://stability.untitledfinancial.com/quote",
        params={"amountUsd": amount_usd, "hasFx": True, "esgScore": 75}).json()
    verified = httpx.get("https://stability.untitledfinancial.com/verify-fees").json()
    return {"quoteId": quote["quoteId"], "totalFee": quote["fees"]["total"]["usd"]}
```
